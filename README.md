## instalação

```bash
pip install requirements.txt
```

## Uso
Os exercícios podem ser executados de duas formas

```bash

#Execute o arquivo main que vai abrir o menu para seleção de exercíos.
python main.py
```

```bash
# Segunda forma execução direta do exercício
```
